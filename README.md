# 🚀 Prompt Engineering with LangChain

###  GenAI Internship Assignment – February 2026

Hi! 
This project is part of my **GenAI Internship Task** where I built a **Mini Prompt Engine** using LangChain.

Instead of using hardcoded prompts, this project focuses on creating **dynamic, reusable, and structured prompt systems** — which is a key concept in real-world Generative AI applications.

---

##  Project Objective

The main goal of this assignment was to:

* Replace hardcoded prompts with reusable templates
* Build dynamic prompt generation systems
* Understand how LLM-based applications use structured prompts

---

## Tech Stack

* Python 
* LangChain
* Google Colab / Jupyter Notebook

---

## Tasks Implemented

###  Task 1: PromptTemplate

Converted static prompts into reusable templates using LangChain.

---

###  Task 2: Multi-Input Prompt System

Built prompts using multiple inputs like:

* Topic
* Audience
* Tone

---

###  Task 3: Prompt Variations Engine

Created different styles of prompts:

* Teaching
* Interview
* Storytelling

---

###  Task 4: ChatPromptTemplate

Implemented chat-based prompts using roles such as:

* Teacher
* Interviewer
* Motivator

---

###  Task 5: Input Validation Layer

Added validation for inputs like audience and tone to ensure correctness.

---

###  Task 6: Prompt Generator App

Developed a function to dynamically generate prompts based on user input and selected style.

---

###  Task 7: Template Reusability Test

Tested how one template can be reused with different inputs.

---

##  Project Workflow

User Input → Validation → Prompt Template → Dynamic Prompt → Output

---

##  Key Learnings

* Importance of prompt structuring in GenAI
* How to design reusable prompt systems
* Working with LangChain PromptTemplate and ChatPromptTemplate
* Building modular and scalable AI pipelines
